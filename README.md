
Local Business Directory 🏪

📌 Project Description

The Local Business Directory is a web application built using Flask that allows users to view and add local businesses. This project demonstrates the use of Object-Oriented Programming (OOP), data structures, and basic web development.

 Objective

The goal of this project is to transform a simple Python program into an interactive web application where users can interact through a browser.

Features
	•	View a list of local businesses
	•	Add new businesses
	•	Simple and user-friendly interface



 Technologies Used
	•	Python
	•	Flask (Web Framework)
	•	HTML (Templates)
	•	OOP (Object-Oriented Programming)

Installation & Setup
	1.	Clone the repository:
git clone <your-repo-link>
2.	Navigate into the project folder:
cd business_directory
	3.	Install Flask:
pip install flask
4.	Run the application:
python app.py
5.	Open in browser:
http://127.0.0.1:5000
